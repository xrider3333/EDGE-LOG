Pipeline stages — plain English
3 · Search — run the strategy hundreds of times with different knobs. Grid = every combo; auto/Bayesian = smart search. Both live.
4 · Pick — today grabs the highest-PnL setting. PDP plateau = a broad hill of good neighbors, not a spike. Ensemble = blend the best several.

Relationship scoring — feature selectioncross-cutting · step 5
Pearson correlationLIVE
▸Mutual Information (MI)PLANNED
WhatScores any dependency between a param/feature and PnL — linear or not. WhyPearson misses inverted-U / threshold relationships (good mid-range, bad at extremes → r ≈ 0). How it's computedsklearn.feature_selection.mutual_info_regression — uses the KSG (Kraskov 2004) k-NN estimator under the hood. KSG is the math inside MI, not a separate score.
▸Predictive Power Score (PPS)PLANNED
WhatA single 0–1 score of how well X predicts Y — asymmetric, nonlinear, handles categoricals. WhyOne readable number to rank params/features; better than r for screening. Tool / caveatppscore. Default model is a tree → use cross-validated PPS so it doesn't overfit tiny samples.

▸PDP plateau selector NEXTPLANNED

▸Adversarial-validation gatePLANNED
WhatTrain a classifier to tell fit-era bars from recent / lockbox bars. If it can, the regime shifted. WhyCatches a champion fit to a stale world. Complements transfer + lockbox. Hooksvalidate.py — one more entry in the checks dict.
▸Conformal band on the gatesPLANNED
WhatWrap the fold / lockbox estimate in a calibrated interval; gate on the lower bound. WhyPoint thresholds ignore small-sample noise; "lower bound > 0" is harder to fool. Toolsvalidate.py gate logic; MAPIE / Venn-ABERS.
▸Causal counterfactual checkPLANNED
What"What's the PnL on these same bars without the signal?" — the counterfactual. WhySeparates a real entry edge from just being long in a rising market. Sharper than the SPY benchmark. Toolsvalidate.py; CausalImpact / interrupted-time-series.
